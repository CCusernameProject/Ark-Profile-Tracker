## Projet SASS Ark

Nom du projet:
```bash
Ark Profile Tracker
```

Technologies employées:
```bash
- React NextJS
# and
- Tailwinds
# and
- Mod ArkDevKit
```

Objectif final:
```bash
Chaque personne possédant un serveur ark pourra installer un mod qui leurs permettra de recensé chaque tribu de leur serveur et avoir toute les possibilités admin. De plus chaque tribu devra disposer d'un coffre fort de tribu (F7 / F8) ou seront déposer les récompenses pour les joueurs. Chaque joueurs de ce même serveur pourra venir consulté toute les informations de sa tribu sur ce site tels que la dernière connexion, les logs de la tribu, les dinosaures actuellement sorti, le nombre de construction avec leurs coordonnées etc. Avec un paywall de 5€ uniquement pour les gérants du serveur tout les 6 mois pour avoir accès au information dite "Confidentiel".
```
